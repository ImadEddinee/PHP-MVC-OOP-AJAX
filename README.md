
# Photo Share
Simple Website to share pictures and interact with other users


## Concepts

The project covers these programming concepts:

* MVC Pattern
* OOP
* PDO
* Asynchronous requests

## Technologies

* HTML
* CSS   *Bootstrap*
* Javascript   *Ajax*
* PHP
* MySql
